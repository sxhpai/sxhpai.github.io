python三种编辑器比较

  1. python 文件，以整体为一块运行（pycharm中的 .py 文件）
    • 优点：通用，传播方便，适用于大型项目
    • 缺点：需要从头运行
  2. python 控制台，以一行为一块运行，或任意行为一块运行
    • 优点：显示每个变量属性
    • 缺点：不利于代码阅读及修改
  3. python jupyter，以任意行为一块运行
    • 优点：利于代码阅读及修改
    • 缺点：环境需要配置

文件配置

  • Dataset: 提供一种方式去获取数据及其label
    • 如何获取每一个数据机器label
    • 告诉我们总共有多少数据
  • Dataloader: 为后面的网络提供不同的数据形式
